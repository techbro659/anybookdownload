# 📚 Telegram Book PDF Finder & Downloader Bot

A powerful Telegram bot that searches multiple free legal book sources and delivers PDFs directly to your chat.

## ✨ Features

- **🔍 Multi-Source Search**: Searches across 8 free book repositories simultaneously
- **⬇️ Direct PDF Downloads**: Downloads books directly to Telegram (up to 15MB)
- **📖 Multiple Formats**: Supports PDF, EPUB, and HTML versions
- **🎯 Interactive Interface**: Easy-to-use inline keyboards for browsing results
- **🌐 Legal Sources Only**: All sources are free and legal
- **⚡ Fast & Responsive**: Optimized for quick searches and downloads

## 📚 Supported Sources

The bot searches the following free book repositories:

| Source | Type | Coverage |
|--------|------|----------|
| **📖 PDF Drive** | Direct PDFs | Modern & classic books |
| **🏛️ Archive.org** | Multiple formats | Millions of texts |
| **📚 Open Library** | EPUB, PDF | Over 1.7M free ebooks |
| **✨ Project Gutenberg** | EPUB, HTML, PDF | 70K+ classic texts |
| **📗 ManyBooks** | Multiple formats | 50K+ free ebooks |
| **⭐ Standard Ebooks** | EPUB, PDF | Carefully curated classics |
| **🔬 Z-Library** | Multiple formats | Largest book collection |
| **📕 Bookboon** | Textbooks | Free educational resources |
